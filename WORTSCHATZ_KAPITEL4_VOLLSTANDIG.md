Catatan revisi: kalimat contoh di bagian B, C, D, F, H sekarang banyak yang pakai Konjunktiv II / irreale Bedingungssätze / Pronominaladverbien, karena bagian-bagian itu memang tentang saran, andaian, dan situasi kerja, jadi grammar-nya nyambung natural. ## M. Redemittel: Sich Entschuldigen & Reagieren

| Deutsch | Indonesia |
|---|---|
| Entschuldigung! / Verzeihung! | Maaf! |
| Entschuldigen/Verzeihen Sie bitte. | Mohon maaf. |
| Das wollte ich nicht. | Itu bukan maksud saya. |
| Das war keine Absicht. | Itu tidak disengaja. |
| Das ist mir wirklich unangenehm/peinlich. | Ini benar-benar canggung/memalukan buat saya. |
| (Es) Tut mir (sehr/schrecklich) leid. | Saya sangat menyesal. |
| Bitte. / Schon gut. | Sama-sama. / Gapapa kok. |
| Kein Problem. | Gak masalah. |
